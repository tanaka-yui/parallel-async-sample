# JavaScript(Node.js)-project

## 環境

- Node 10+

## 実行方法
```sh
npm run main
```

## 実行結果
```sh
[JavaScript] main start
test1 method
test2 method
123
456
[JavaScript] main end
```

## おまけ

`main2.js`はcallback関数で出来たライブラリをasync/awaitに変えたい時に使うPromiseのサンプルです。
kotlinプロジェクトの方でも同じ様にcallbackをcoroutineでawaitっぽくできる様にするサンプルがあります。
