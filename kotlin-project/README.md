# Kotlin-project

## 環境

- Android Studio 3.5.x

## 実行方法
app/src/main/java/sample/kotlin/Main.kt
を右クリックしてRunで実行できます。

## 実行結果
```sh
[kotlin] main start
test1 method
test1 method
123
456
[kotlin] main end
```

## おまけ

`Main2.kt`はcallback関数で出来たライブラリをasync/awaitに変えたい時に使うサンプルです。
JavaScriptのnew Promiseど同様な使い方ができます。
